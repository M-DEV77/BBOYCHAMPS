# B.BOY CHAMPS
#### Descrição:
~~~
Aplicação para criar cha de campeonato de Break(BBoy)
com o intuito de facilitar a execução de campeonatos,
desde da inscrição ao julgamento execução das chaves
eo processo de votação dos competidores. 
~~~
